## scripter

#### scripting language

### Usage:
```
scripter [OPTIONS] script_file
```

Command Basics:
- `$` : define a variable
- `@` : run a os routine
- `-` : execute a program and wait for it to finish
- `?` : conditional
- `%` : goto
- `>` : user input
- `#` : line number label

Syntax:
`$` statement: define or set a variable
```
$variable_name [expression]
```

`@` statement: run a os routine
```
@routine_addr [REG=value, [REG2=value2, etc.,]]
```
